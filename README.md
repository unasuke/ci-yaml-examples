# CI yaml examples

## What's?
It's my CI config YAML template repository.

## How to use
Plase yaml file to specific path and tweak this.

| name | path |
| -- | -- |
| `travis.yml` | `.travis.yml` |
| `circleci.yml` | `.circleci/config.yml` |
| `github.yml` | `.github/workflows/foo.yml` |
